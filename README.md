# i3-config
Config for an i3-gaps setup

### Installation
Install the dependencies, put the contents of the config folder into /home/.config

### Dependencies: 
- i3-gaps
- i3status
- pasystray
- nm-applet
- feh
- picom
- spectacle
- rofi
- polkit-gnome

### Wallpaper
To change the wallpaper, put it in .config/Wallpapers/Wallpaper.jpg.
